  • Q: Who built Concordia's Art Management System?
  • A: This system was built by Jason Rametta, Akshar Patel and Oscar Bobadilla.

  • Q: How was this system built?
  • A: We used many technologies. The site's backend is written in Python and built on the Django framework. Data is stored in a relational database powered by PostgreSQL. The site is hosted on Heroku servers and media is stored on Amazon's AWS S3 server. The frontend is built upon the Bootstrap framework and the jQuery library. All code was written in the PyCharm IDE and code revisions were tracked on our Github page. Languages used include: Python, JavaScript, SQL, HTML & CSS.

  • Q: Does the system work on mobile phones?
  • A: Yes! We follow the latest trends in web development and have 100% compatibility with all smart phones and tablets.

  • Q: I added my work but I can't see it, where did it go?
